Important FAQ

What is the Document Object Model(DOM)?

If we closely observe the HTML document, we will find that it is structured as an object in JavaScript. This structured representation is called the Document Object Model (DOM). Since it is similar to a JavaScript object, we can easily create, access, and manipulate HTML elements through JavaScript. The DOM allows us to interact with and dynamically modify a webpage's content, structure, and style.

How do you select element from the DOM?

To select elements from the DOM in JavaScript, we can use various methods. Some of those are: getElementById(), getElementsByClassName(), getElementsByTagName(), querySelector(), querySelectorAll() etc.

What is the event delegation in the context of DOM, and why it is useful?

When we want to handle higher level element we have use a technique using bubble effect in DOM. This technique is called event delegation in the context of DOM. It is very useful when we want to dynamically add child from parent.

How do you manipulates an element's attributes and styles using the DOM?

We can manipulate an element's attributes and styles in bunch of different methods. Some of those are: getAttribute(), setAttribute(), removeAttribute(), and setting value of propery using elemnt.style.respective-style.
